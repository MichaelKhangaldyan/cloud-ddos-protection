# v1 — Բնօրինակ տարբերակ

Այս թղթապանակում գտնվում են ավարտական աշխատանքի **բնօրինակ** կոդի ֆայլերը,
որոնք համապատասխանում են դիպլոմի տեքստին (Գլուխ 2):

## Ֆայլեր

| Ֆայլ | Նկարագրություն |
|------|----------------|
| `auto_block.py` | Ավտոմատ IP արգելափակման սկրիպտ |
| `dashboard_original.py` | Flask Monitoring Dashboard |

## Աշխատանքի սկզբունքը (v1)

```
1. cron-ով յուրաքանչյուր րոպե գործարկվում է auto_block.py
2. Սկրիպտը կարդում է /var/log/nginx/attack.log
3. Հաշվում 429 պատասխանները ըստ IP-ի
4. THRESHOLD = 10 գերազանցելու դեպքում՝
   - ավելացնում է deny IP կանոնը /etc/nginx/blocked_ips.conf-ում
5. Reload-ում է Nginx-ը
6. Այսուհետ տվյալ IP-ն ստանում է 403 Forbidden
```
